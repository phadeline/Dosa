# Dosa

## Description
The purpose of this project was to create a python script that allows a restaurant to organize/get a list of their customers with their numbers and a list of items orders along with the items price and the total amount of time the item was ordered. This is accomplished by writing a function that takes a json file as an argument. The contents of the file is read and the desired information is then written on two specific files (customers.json and items.json).

## User Story

- AS a restaurant owner
- I WANT to have a list of my customers with their phone numbers
- I WAnt to have a list of items orders with their prices and total amount of times ordered
- SO THAT I can keep a record of my customers and items

## Acceptance Criteria

- Given a Json file with data
- When I open the file
- Then I will read it and create 2 json files: one for customers and their number and another one for items with prices and total times ordered
- When I view the created json files
- Then I am presented with the correct information with no duplicates
